Inicio Rápido

1 Versión Básica (Sin Configuración)

La forma más simple de instalar Easynav. Solo agrega el script y funciona automáticamente. 

<!-- Easynav Free - Versión Básica -->
<script src="https://cdn.easynav.cl/easynav.min.js" defer></script>

2 Versión Personalizable

Usa la versión custom para personalizar colores, comportamiento y más. 

<!-- Easynav Free - Versión Personalizable -->
<script src="https://cdn.easynav.cl/easynav-custom.min.js" defer></script>

2. Configuración via JavaScript

Define un objeto de configuración antes de cargar el script para mayor control. 

<script>
  window.EasynavConfig = {
    primaryColor: '#059669',
    backgroundColor: '#ecfdf5',
    fontSizeMin: 70,
    fontSizeMax: 200,
    fontSizeStep: 15,
    floatingPosition: 'top-left',
    onFontChange: function(size) {
      console.log('Tamaño cambiado a:', size);
    }
  };
</script>

Métodos de la API

Controla Easynav programáticamente usando la API global window.Easynav.

.increaseFontSize()

Aumenta el tamaño de fuente por el valor de step.

// Aumentar tamaño
Easynav.increaseFontSize();

// Encadenar métodos
Easynav.increaseFontSize().increaseFontSize();

.decreaseFontSize()

Disminuye el tamaño de fuente por el valor de step.

Easynav.decreaseFontSize();

.setFontSize(size)

Establece un tamaño de fuente específico (se limita a min/max).

// Establecer al 120%
Easynav.setFontSize(120);

// Establecer al máximo
Easynav.setFontSize(150);

.toggleContrast()

Alterna el modo de alto contraste.

Easynav.toggleContrast();

.setContrast(enabled)

Establece el estado del alto contraste.

// Activar contraste
Easynav.setContrast(true);

// Desactivar contraste
Easynav.setContrast(false);

.reset()

Restablece todas las configuraciones a valores por defecto.

Easynav.reset();

.getState()

Obtiene el estado actual (fontSize, highContrast, etc.).

const state = Easynav.getState();
console.log(state);
// { fontSize: 100, highContrast: false, isFloating: false }

.setOptions(options)

Actualiza las opciones en tiempo de ejecución.

// Cambiar el color primario en runtime
Easynav.setOptions({
  primaryColor: '#dc2626'
});

.destroy()

Destruye la instancia y limpia el DOM.

Easynav.destroy();

Callbacks / Eventos

Escucha cambios y reacciona a las acciones del usuario.

onInit

Se ejecuta cuando Easynav se inicializa.

window.EasynavConfig = {
  onInit: function(instance) {
    console.log('Easynav inicializado!');
  }
};

onFontChange

Se ejecuta cuando cambia el tamaño de fuente.

window.EasynavConfig = {
  onFontChange: function(newSize, instance) {
    console.log('Nuevo tamaño:', newSize + '%');
    // Ejemplo: enviar analytics
    gtag('event', 'font_change', { value: newSize });
  }
};

onContrastChange

Se ejecuta cuando cambia el estado del alto contraste.

window.EasynavConfig = {
  onContrastChange: function(isEnabled, instance) {
    console.log('Alto contraste:', isEnabled ? 'ON' : 'OFF');
  }
};

onReset

Se ejecuta cuando se restablece la configuración.

window.EasynavConfig = {
  onReset: function(instance) {
    console.log('Configuración restablecida');
  }
};

onFloatingChange

Se ejecuta cuando cambia el estado flotante.

window.EasynavConfig = {
  onFloatingChange: function(isFloating, instance) {
    console.log('Modo flotante:', isFloating);
  }
};
